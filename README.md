# Company API

This project is a .NET 7 Web API that demonstrates CRUD operations,
database relationships, and reporting queries using Entity Framework Core
and SQL Server.

---

## Technologies Used
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- LINQ
- Swagger

---

## Database Setup

1. Open SQL Server Management Studio (SSMS)
2. Execute the SQL scripts in the following order:
--Database/CreateDatabase.sql
--Database/CreateTables.sql
--Database/InsertSampleData.sql


## API Setup

1. Clone the repository:
https://github.com/Ahmedabdelfatah11/CFM_Task.git
